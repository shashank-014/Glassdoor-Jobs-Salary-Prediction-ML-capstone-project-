# Glassdoor Jobs Salary Prediction

I built this project to turn a messy Glassdoor salary file into something I could actually explain clearly. I wanted the work to feel practical, so I kept the feature set readable and focused on what usually drives salary bands in job posts.

## How I worked on it

I kept the flow simple and presentation friendly:
- I start with a quick raw data check.
- I clean only the fields I really need.
- I move into analysis, modeling or clustering, then I wrap it up with a short takeaway.

## Deliverables

- app.py : this is the walkthrough version I can open in Streamlit.
- Glassdoor_Jobs_Salary_Prediction.ipynb : this is the main capstone notebook in my own writing style.
- glassdoor_jobs_project_code_export.py : this is the reusable code export behind the notebook flow.

## Data Files

- glassdoor_jobs.csv

## How I run it

1. python -m venv .venv
2. .venv\Scripts\Activate.ps1
3. pip install -r requirements.txt
4. streamlit run app.py
